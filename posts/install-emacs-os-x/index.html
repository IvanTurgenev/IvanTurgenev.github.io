install emacs and doom-emacs on os x catalina with brew

2020-03-26 :: me
#install  #os x  #brew  #emacs 

You have already installed brew, now you install the emacs cask.

brew cask install emacs

Next open and close emacs so it creates ~/.emacs.d, then install doom-emacs.

git clone https://github.com/hlissner/doom-emacs ~/.emacs.d
~/.emacs.d/bin/doom install

After install, configure and enable modules you need. Put this on your ~/.doom.d/config.el so you will be able to insert special characters with the right option key.

;; I need my Spanish ñ.
(setq mac-right-option-modifier nil)
